### UV setup
If `uv` is not installed, install it. <br>
https://docs.astral.sh/uv/ <br>
```
# install uv and python if not installed
curl -LsSf https://astral.sh/uv/install.sh | sh
uv python install
```
### Make environment
make environment with
```
make install
```
